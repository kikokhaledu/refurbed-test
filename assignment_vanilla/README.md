# Vanilla Track Overview

This folder contains the vanilla JavaScript starter track.

## Structure
```text
assignment_vanilla/
`-- frontend-vanilla/    # Vanilla JS + Vite starter
```

Backend is shared from the repository root in `backend/`.

## Local Development
1. Start backend:
```bash
cd backend
go run .
```
2. Start vanilla frontend:
```bash
cd assignment_vanilla/frontend-vanilla
npm install
npm run dev
```

## URLs
- Frontend: `http://localhost:5173`
- Backend: `http://localhost:8080`
