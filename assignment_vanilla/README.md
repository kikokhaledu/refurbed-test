# Vanilla JavaScript Assignment

## Structure

```
assignment_vanilla/
└── frontend-vanilla/    # Vanilla JS + Vite frontend
```

**Backend:** Use the shared `backend/` folder at the root level.

## Quick Start

### 1. Start Backend (from root)
```bash
cd ../backend
go run main.go
```

### 2. Start Frontend
```bash
cd frontend-vanilla
npm install
npm run dev
```

Frontend runs on: `http://localhost:5173`  
Backend runs on: `http://localhost:8080`
